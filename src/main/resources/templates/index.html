LiteManagerX

Lite Manager X（简称 LMX）是一个用于统一接入、规范管理和集中处理请求的轻量级管理平台，主要解决多接口、多模块场景下请求分散、命名混乱、维护成本高的问题。

项目简介

LiteManagerX 是一个基于 Spring Boot 构建的轻量级 Web 管理服务， 旨在提供统一、直观的数据库信息查询接口。 项目专注于"简单、清晰、可维护"， 通过 REST API 的形式快速获取数据库结构与数据内容， 适合作为内部工具、学习项目或二次开发的基础框架。

核心功能

数据库列表查询

通过统一接口获取当前 MySQL 实例中的所有数据库名称， 快速掌握整体数据结构。

表结构信息

基于 INFORMATION_SCHEMA， 动态查询指定数据库下的表与字段信息， 包括字段类型、主键信息等。

表数据访问

支持直接读取指定表中的全部数据， 方便调试、学习和快速验证数据内容。

轻量 & 可扩展

不依赖复杂中间件， 代码结构清晰，便于后续扩展权限控制、 前端界面或更多数据源支持。
