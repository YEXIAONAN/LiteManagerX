项目简介

LiteManagerX（LMX）是一款基于 Spring Boot 的轻量级 Web 管理服务，专注“简单、清晰、可维护”， 通过 REST API 一键获取 MySQL 结构与数据，适合内部工具、教学示例或二次开发。

核心功能

数据库列表

实时获取当前实例所有库名，结构一目了然。

表结构查询

基于 INFORMATION_SCHEMA 动态生成字段清单与主键信息。

数据预览

零 SQL 直接浏览表数据，调试效率翻倍。

轻量 & 可扩展

无冗余依赖，代码清晰，可快速植入权限、缓存、多数据源。

技术栈

Spring Boot 3 JdbcTemplate MySQL 8 Glassmorphism UI Maven
